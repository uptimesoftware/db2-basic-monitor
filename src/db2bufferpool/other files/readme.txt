DB2 Buffer Pool Plug-In Monitor

This plug-in monitor collects statistics for all of the buffer pools defined on the DB2 instance, including read and write statistics, and hit ratio statistics.  You can then alert based on pre-defined thresholds.

Buffer pool statistics collected include the following read/write times and ratios:
- Non-Accessed Asynchronous Reads
- Average Asynchronous Read Time
- Average Synchronous Read Time
- Average Read Time
- Non-Prefetched Synchronous Reads
- Average Asynchronous Write Time
- Average Synchronous Write Time
- Average Write Time
- Asynchronous Write Percentage
- Data Hit Ratio
- Index Hit Ratio
- Auxiliary Storage Objects Hit Ratio
- Total Hit Ratio (Index, Data and XDA)
